# 🎮 Tic Tac Toe Game

A classic **Tic Tac Toe** game built with HTML, CSS, and JavaScript.

## 🚀 Features
- 2-player mode
- Interactive UI
- Game restart option
- Win/draw detection

## 💻 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
